## 커밋 메시지 규칙

| 키워드     | 설명                             |
|------------|----------------------------------|
| `feat`     | 새로운 기능이나 코드 추가 시     |
| `fix`      | 버그 수정 시                     |
| `docs`     | 문서 수정 시                     |
| `style`    | 코드 형식 변경 (기능 변경 없음)  |
| `refactor` | 코드 리팩토링 시                 |
| `test`     | 테스트 코드 작성 시              |
| `chore`    | 기타 작업 시                     |
| `ci`       | CI/CD 파이프라인 변경 시         |
| `pull`     | 브랜치 병합 시                   |
| `merge`    | PR 머지 시                       |
| `delete`   | 파일 삭제 시                     |

### 예시

```agsl
feat: 회원 가입 기능 구현

회원 가입 기능을 구현하였습니다.

Resolves: #123
Ref: #456
Related to: #48, #45
```
